* Common Mistakes

** âœ— Mistake: Setting both fixed days and gradient quality

#+begin_src elisp
;; gradient-quality is IGNORED when using fixed days
(setq blame-reveal-gradient-quality 'strict      ; â† No effect!
      blame-reveal-recent-days-limit 30)         ; â† Only this is used
#+end_src

/Why?/ ~gradient-quality~ only controls the /auto-calculation strategy/.
When you specify a fixed number (30, 90, 180), there's no calculation to control.

** âœ“ Correct: Choose one approach

#+begin_src elisp
;; Approach 1: Let auto mode calculate optimal window
(setq blame-reveal-recent-days-limit 'auto
      blame-reveal-gradient-quality 'strict)
;; â†’ Calculates ~20-40 days with excellent color distinction

;; Approach 2: Use fixed window (gradient-quality not needed)
(setq blame-reveal-recent-days-limit 30)
;; â†’ Always 30 days, shows all commits in that period
#+end_src

** When to use which?

| Choose Fixed Days (30, 90, 180) | Choose Auto + Quality |
|--------------------------------+-----------------------|
| Want predictable time window   | Want optimal colors   |
| Know your team's rhythm        | Files vary in activity |
| "Show me last month"           | "Show me ~10 commits" |

#+begin_src elisp
;; Example 1: Predictable window
(setq blame-reveal-recent-days-limit 90)
;; Active file: 50 commits in 90 days (colors may be subtle)
;; Quiet file: 3 commits in 90 days (excellent distinction)

;; Example 2: Consistent quality
(setq blame-reveal-recent-days-limit 'auto
      blame-reveal-gradient-quality 'strict)
;; Active file: ~30 days, 8 commits (excellent distinction)
;; Quiet file: ~180 days, 8 commits (excellent distinction)
#+end_src

* Quick Reference

| Want                  | Setting                                              |
|-----------------------+------------------------------------------------------|
| Smart defaults        | ~'auto + 'auto~                                        |

| More history          | ~'auto + 'relaxed~ or ~180~ (days)                       |
| All commits           | ~nil~                                                  |
| Green colors          | ~:hue 120~                                             |
